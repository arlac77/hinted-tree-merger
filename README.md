# hinted-tree-merger

merges two trees guided with hints

# usage

```js
import { mergeArrays } from "hinted-tree-merger";

const r = mergeArrays(
        [{ k:1 e:1},{ k:2}],
        [{ k:1 e:2},{ k:3}], // 2nd. array has precedence
        "",
      undefined,
      {
        "": { key: "k" } // identify slots by property "k"
      });

  // r := [{ k:1 e:2 }, { k:2 }, { k:3 }] 
```
